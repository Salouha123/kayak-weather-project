#  Projet Kayak Last Minute

##  Description
Système de recommandation de destinations basé sur la météo en France. 
Le projet analyse les prévisions météo de 35 villes françaises et recommande les 5 meilleures destinations.

##  Fonctionnalités
- Géocodage automatique des villes
- Analyse des prévisions météo sur 7 jours
- Calcul de score météo intelligent
- Sélection des Top 5 destinations
- Cartes interactives avec Plotly
- Recommandations d'hôtels
